# 日記プラットフォーム

シンプルな日記投稿プラットフォームです。

## 機能

- 日記の作成・編集・削除
- カテゴリー分類
- 検索機能
- マークダウン対応
- 画像アップロード

## セットアップ

```bash
cd diary-platform
pip install -r requirements.txt
python app.py
```

## 技術スタック

- Backend: Flask/FastAPI
- Frontend: HTML/CSS/JavaScript
- Database: SQLite/PostgreSQL
- Storage: ローカル/S3
